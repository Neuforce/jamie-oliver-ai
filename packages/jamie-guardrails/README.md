# jamie-guardrails

Shared Jamie Oliver AI guardrails: NeuGate client, query gate, policy JSON, and PrePrompt rendering.

Used by `apps/backend-search` and `apps/backend-voice`.

```bash
pip install -e packages/jamie-guardrails
```

Policy file: `jamie_guardrails/data/jamie-policy.json` (override with `JAMIE_POLICY_PATH`).
